Developer Documentation — Add Your API to the Platform

Welcome to the Asstgr platform. This documentation explains how to integrate your APIs.

📡 Public API — Endpoints Reference

Asstgr provides a full REST API to manage your APIs programmatically. All endpoints require authentication via an API Key.

🔐 Authentication:
Include your API key in the request headers: 
Authorization: Api-Key YOUR_API_KEY

🔑 API Keys

Method URL Description
GET/api/v1/keys/List your API keys
POST/api/v1/keys/Create a new API key
DELETE/api/v1/keys/{id}/Revoke an API key

📊 Quota & Limits

Method URL Description
GET/api/v1/quota/Get current usage and remaining quota
GET/api/v1/limits/Get platform limits

📚 APIs

MethodURLDescription
GET/api/v1/apis/List your APIs
POST/api/v1/apis/Create a new API
GET/api/v1/apis/{api_id}/Retrieve API details
PATCH/api/v1/apis/{api_id}/Update an API
DELETE/api/v1/apis/{api_id}/Delete an API

🔗 Endpoints

MethodURLDescription
GET/api/v1/apis/{api_id}/endpoints/List endpoints
POST/api/v1/apis/{api_id}/endpoints/Create endpoint
GET/api/v1/apis/{api_id}/endpoints/{id}/Get endpoint
PATCH/api/v1/apis/{api_id}/endpoints/{id}/Update endpoint
DELETE/api/v1/apis/{api_id}/endpoints/{id}/Delete endpoint

⚙️ Parameters

MethodURLDescription
GET/api/v1/apis/{api_id}/endpoints/{endpoint_id}/parameters/List parameters
POST/api/v1/apis/{api_id}/endpoints/{endpoint_id}/parameters/Create parameter

📨 Headers

MethodURLDescription
GET/api/v1/apis/{api_id}/endpoints/{endpoint_id}/headers/List headers

MethodURLDescription

MethodURLDescription

⚙️ API Limits & Rate Limiting

📦 Resource Limits

ResourceLimit
APIs per account100
Endpoints per API10
Parameters per endpoint15
Headers per endpoint10
API keys per account5

🚦 Rate Limiting (Throttling)

Your API is protected by two levels of rate limiting:

  • Burst limit — prevents too many requests in a short time
  • Sustained limit — controls long-term usage
⚠️ Note:
Rate limits are applied per API key.
If exceeded, requests will be temporarily blocked with a 429 Too Many Requests response.

📊 Quota System

Each API call consumes quota credits based on your plan. You can monitor your usage via: 

GET /api/v1/quota/

The response includes:

  • Used calls
  • Remaining calls
  • Monthly limit
  • Usage percentage
